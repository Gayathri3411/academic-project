## 🚀 How to Run the Project

1. Clone or download the project.
2. Start your local server (XAMPP/WAMP).
3. Create the database and import `women_application.sql`.
4. Place the project in `htdocs/` or equivalent.
5. Visit `http://localhost/women_legal_advocate_app/index.php`.
